def to_degree(v):
    d = v // 10000000
    m = (v - d * 10000000) // 100000
    s = v - d * 10000000 - m * 100000
    return d + m / 60 + s / (3600 * 1000)

  • 各事故地点について、それが含まれる校区を結合する。

    • ほぼ差はないので、警告を無視してそのまま進めても構わない。
